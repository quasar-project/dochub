Подробное описание

Предоставляет тривиальную конфигурацию логгера

Предоставляет конфигурацию логгера, которая требует минимальной настройки. Логгер пишет в файл с заданным именем, ограниченный размером и максимальным количеством файлов.

Заметки
Автор библиотеки не советует использовать эту конфигурацию для своих логгеров. Основное предназначение этого класса - быстрая настройка логгера для отладки (например, для тестов).
